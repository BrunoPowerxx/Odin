About This Project

This is a demonstration Django website built to showcase a multi-page application structure that's ready for deployment to Render.com or any other hosting platform.

Features

  • Clean, responsive design with custom CSS
  • Multiple interconnected pages with navigation
  • Blog functionality with Django models
  • Admin interface for content management
  • Static file handling with WhiteNoise
  • Environment-based configuration
  • PostgreSQL support for production
  • Ready-to-use build script for Render

Technology Stack

  • Django 4.2
  • Python 3.11
  • Gunicorn (WSGI server)
  • WhiteNoise (Static files)
  • PostgreSQL (Production database)
